# I Ching Skill · 易经占卜

一个铁口直断、以卦爻为唯一依据的 **AI 易经占卜师**。  
适用于 Openclaw 等支持 Skill 机制的 AI 工具。

## 简介

这不是一本泛泛的人生指南，而是一个**不说废话的占卜师**。

三条铁律：
- 每句话都必须锚定在具体卦爻上——说不出一句通用鸡汤
- 结论永远硬、永远短——吉就吉，凶就凶，能做就做，不能做就直说
- 用户问一句，卦就说一句——不追问背景，不脑补情节

## 安装

**方式一**

```bash
npx skills add @fxckai/iching
```

**方式二：下载手动安装**

1. 将本仓库完整下载（含 `SKILL.md` 和 `scripts/`）
2. 放入你所用 AI 工具的 skills 目录（如 Claude Code 的 `~/.claude/skills/`、Openclaw 的 `.openclaw/workspace/skills/`）
3. 对话中输入关键词 **"算卦""起卦""易经""周易""I Ching"** 等，或手动 `/iching`

> **依赖**：起卦脚本 `scripts/cast_hexagram.py` 需要 Python 3 环境，无第三方依赖。

## 核心设计

### 随机起卦 — 一卦一爻

64 卦中随机取一卦，6 爻中随机取一动爻。每次调用 `cast_hexagram.py`，输出 JSON 格式的卦名、卦序、上下卦、爻位与爻名。

### 断卦五步

```
得卦 → 爻辞 → 断 → 建言 → 自查
```

每一段都短，每一句话都有来处。

### 结论即答案

| 问法 | 回答 |
|:---|:---|
| 「能不能 / 行不行」 | **能** / **不能** |
| 「要不要 / 该不该」 | **要做** / **不做** |
| 「会不会 / 能不能」 | **会** / **不会** |
| 运势类（事业运 / 财运等） | **吉** / **平** / **凶** |

### 反巴纳姆过滤器

输出完成后逐句自查：
- 这句话对另一个完全不相关的问题也适用？→ 删掉
- 这句话不是来自卦名、卦象或爻辞原文？→ 删掉
- 用了「冷静思考」「认真对待」「好好把握」这类万能词？→ 删掉

## 使用示例

激活 Skill 后，你可以这样提问：

- 「我最近考虑跳槽去一家新公司，对方薪资高 30%，但行业方向不太一样。帮我起一卦看看该不该去？」
- 「和女朋友谈了三年，越来越没话说。我们还能走下去吗？」
- 「下个月考研，复习了半年但总感觉没复习到。学业运怎样？」
- 「借给发小 20 万，拖了一年不回消息。他会不会主动还钱？」

Skill 会直接起卦、给出卦名和爻辞、下一句硬结论，而不是绕弯子的「建议你综合考虑」。
